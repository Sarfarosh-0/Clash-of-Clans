# Clash of Clans Landing Page

A clean, responsive, and modern landing page themed around **Clash of Clans**. Built using HTML5, Tailwind CSS, and a touch of vanilla JavaScript, this page features fluid layouts, animations, and an interactive mobile navigation menu.

---

## 🌐 Live Preview

You can view the live, interactive webpage here:
👉 **[Live Demo Link](https://sarfarosh-0.github.io/Clash-of-Clans/)**

## ✨ Features

*   📱 **Fully Responsive:** Optimized for desktop, tablet, and mobile devices.
*   ✨ **Smooth Animations:** Includes subtle hover effects, interactive card scaling, and modern scroll-driven fade-in animations.
*   🍔 **Mobile Menu:** Lightweight vanilla JS sidebar navigation for smaller screens.
*   🎨 **Modern Design:** Uses Tailwind CSS utility classes to achieve a polished, cohesive game theme.

---

## 🛠️ Built With

*   **HTML5** – Structured semantic elements.
*   **Tailwind CSS** – Rapid, responsive styling and animations.
*   **Vanilla JavaScript (ES6)** – Lightweight DOM manipulation for mobile menu toggles.
