# Clash of Clans Landing Page

A responsive **Clash of Clans** themed landing page built using **HTML5** and **Tailwind CSS**. The project features a modern UI with sections for game features, heroes, troops, game statistics, and a download call-to-action.

## 🚀 Features

* Responsive design
* Modern UI with Tailwind CSS
* Hero and troop showcase
* Game features section
* Download section
* Smooth hover animations

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* Google Fonts

## 📂 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/clash-of-clans-landing-page.git
   ```

2. Open `index.html` in your browser.
